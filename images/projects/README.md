# Project Images Directory

This directory contains images and screenshots for the portfolio projects.

## Network Scanner Pro Images

Add the following images to `/images/projects/network-scanner/`:

### Recommended Screenshots:
1. **main-gui.png** - Full GUI application window showing:
   - Gradient header with "🔍 Network Scanner Pro" title
   - Card-based layout with 6 main sections
   - Network preset buttons (192.168.1.0/24, 10.0.0.0/24, 172.16.0.0/24)
   - Port configuration with range inputs
   - 2x2 scan operation button grid
   - Progress bar with percentage
   - Color-coded results area with timestamps

2. **cli-demo.png** - Terminal screenshot showing CLI commands:
   ```
   ./network-scanner-cli ping google.com
   ./network-scanner-cli portscan 192.168.1.1 1 1000
   ./network-scanner-cli netscan 192.168.1.0/24
   ```

3. **architecture-diagram.png** - Visual representation of the layered architecture
4. **performance-chart.png** - Performance metrics visualization
5. **cross-platform-builds.png** - GitHub Actions build pipeline

### Image Specifications:
- **Format**: PNG preferred for screenshots, JPG for photos
- **Resolution**: Minimum 1200px width for main screenshots
- **Quality**: High quality, clear text visibility
- **Privacy**: Blur any sensitive IP addresses or network information

### Usage in Portfolio:
These images will be automatically displayed in the project showcase sections. Update the HTML img tags with the appropriate paths when images are available.
